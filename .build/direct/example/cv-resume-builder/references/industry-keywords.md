## Using Keywords Effectively

### Keyword Density
- Include 10-15 relevant keywords per role
- Natural integration > keyword stuffing
- Match job posting terminology exactly

### Placement Strategy
1. **Skills section**: Technical tools and certifications
2. **Experience bullets**: Methodologies and practices
3. **Summary**: Domain expertise and specializations

### Variation Tips
- Use both acronyms and spelled-out forms: "SEO (Search Engine Optimization)"
- Include tool versions when relevant: "Python 3.x", "React 18"
- Add proficiency levels for languages: "(fluent)", "(native)"
