## Compound Formation Rules

### Latin Compounds

```
Genitive + Nominative = "X of Y"
artium (of arts) + codex = codex of arts

Adjective + Noun = "X-ish Y"
magnus + opus = great work

Noun + Noun = "X-Y" (first modifies second)
ars + factor = skill-maker
```

### Greek Compounds

```
Genitive + Nominative = "X of Y"
technōn (of arts) + biblos = book of arts

Root + Root = compound meaning
technē + logos = study of arts (technology)

Prefix + Root = modified meaning
syn + thesis = putting together (synthesis)
```
