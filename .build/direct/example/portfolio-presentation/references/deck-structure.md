# Portfolio Deck Structure

Guidelines for creating presentation-style portfolio decks.

## When to Use Deck Format

| Best For | Avoid For |
|----------|-----------|
| Live presentations | Self-paced review |
| Client pitches | Detailed documentation |
| Interview presentations | Email attachments |
| Team reviews | Print distribution |
| Video recordings | SEO purposes |

## Presentation Tips

### Timing

| Slide Type | Target Time | Notes |
|------------|-------------|-------|
| Title | 30 seconds | Quick intro |
| About | 1 minute | Brief positioning |
| Project overview | 30-60 seconds | Hook interest |
| Deep dive | 2-3 minutes | Main content |
| Results | 1 minute | Let metrics sink in |
| Contact | 30 seconds | Clear next steps |

### Total Presentation

- Quick version: 10-15 minutes
- Standard version: 20-30 minutes
- Deep dive: 45-60 minutes

### Narrative Flow

```
1. Hook: Start with your most impressive result
2. Context: Brief background
3. Build: Show progression through projects
4. Peak: Your best/most relevant work
5. Land: Clear call to action
```

---

## Design Guidelines

### Visual Consistency

```
Elements to standardize:
- Title placement and size
- Image frame style
- Metric callout format
- Color palette
- Typography scale
- Spacing/margins
```

### Color Usage

```
Primary: Hero backgrounds, section headers
Secondary: Accents, metrics
Text: Dark gray (#333) on light backgrounds
Inverse: White text on dark backgrounds
```

### Animation (If Used)

```
DO:
- Subtle fade-ins (0.3s)
- Sequential reveals for lists
- Smooth transitions between slides

DON'T:
- Spinning/flying elements
- Sound effects
- Heavy motion
- Distracting transitions
```

## Platform-Specific Tips

### Figma/FigJam

```
Export: PDF (all pages)
Present: Prototype mode
Share: View-only link
```

### Google Slides

```
Export: PDF or PowerPoint
Present: Present from beginning
Share: Anyone with link can view
```

### PowerPoint

```
Export: PDF, PowerPoint, or video
Present: Slideshow mode
Compatibility: Test on target system
```

### Keynote

```
Export: PDF, PowerPoint, or Keynote
Present: Play slideshow
Magic Move: Use for smooth transitions
```
