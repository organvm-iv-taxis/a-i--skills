# Color Harmony Reference

Color theory foundations for design-forward canvas work.

## Color Relationships

### Primary Harmonies

**Complementary**: Opposite on color wheel
```
Blue ←→ Orange
Red ←→ Green
Yellow ←→ Purple
```
High contrast, vibrant, use one dominant + one accent

**Analogous**: Adjacent on color wheel
```
Yellow → Yellow-Orange → Orange
Blue → Blue-Green → Green
```
Harmonious, natural, low contrast

**Triadic**: Evenly spaced (120° apart)
```
Red ↔ Yellow ↔ Blue
Orange ↔ Green ↔ Purple
```
Vibrant, balanced, use one dominant

**Split-Complementary**: Base + two adjacent to complement
```
Blue + Red-Orange + Yellow-Orange
```
High contrast but more nuanced than complementary

### Extended Harmonies

**Tetradic (Rectangle)**: Two complementary pairs
```
Red + Green + Blue + Orange
```
Rich, requires careful balance

**Square**: Four colors evenly spaced
```
Red + Yellow-Green + Blue + Red-Violet
```
Dynamic, best with one dominant

## Color Properties

### Hue, Saturation, Brightness

```
HSB Model:
H = Hue (0-360°): The color itself
S = Saturation (0-100%): Intensity/purity
B = Brightness (0-100%): Light/dark
```

### Creating Palette Variations

**Tints**: Add white (increase brightness, decrease saturation)
**Shades**: Add black (decrease brightness)
**Tones**: Add gray (decrease saturation)

```
Base color: #3498db (blue)
Tint:       #85c1e9
Shade:      #1a5276
Tone:       #6699aa
```

## Palette Construction

### Monochromatic

One hue, varied saturation/brightness:
```
Dark:   #1a252f
Medium: #34495e
Light:  #5d6d7e
Accent: #85929e
```

### Warm Palette

Red, orange, yellow, brown:
```
#c0392b (red)
#e74c3c (light red)
#d35400 (orange)
#f39c12 (yellow-orange)
```

### Cool Palette

Blue, green, purple:
```
#2980b9 (blue)
#3498db (light blue)
#1abc9c (teal)
#9b59b6 (purple)
```

### Neutral Palette

Grays with subtle undertones:
```
Cool gray:  #2c3e50
Warm gray:  #5d5c61
True gray:  #95a5a6
Off-white:  #ecf0f1
```

| Level | Ratio | Use |

| AAA Large | 4.5:1 | Large text |

### Color Blindness Considerations

Avoid relying solely on:
- Red/green distinction (8% of men)
- Blue/yellow distinction (rare)
- Saturation differences alone

Safe combinations:
- Blue and orange
- Blue and yellow
- Dark and light (value contrast)

## Quick Palette Tools

### From a Photo

1. Identify dominant color
2. Find 2-3 supporting colors
3. Add neutral (white/black/gray)
4. Test contrast ratios

### From Scratch

1. Choose base hue for mood
2. Add complement or split-complement
3. Create tints/shades for depth
4. Designate neutral background
5. Reserve brightest for accent

### Hex Adjustments

```
Lighten: Increase each RGB value
Darken:  Decrease each RGB value
Saturate: Increase distance from gray
Desaturate: Move toward gray

#3498db (base)
#5dade2 (lighter: +41, +21, +7)
#2471a3 (darker: -16, -39, -56)
```
