# Composition Principles

Foundational principles for creating visually compelling canvas designs.

## The Grid

### Rule of Thirds

Divide canvas into 9 equal parts. Place key elements at intersections.

```
┌───────┬───────┬───────┐
│       │   ●   │       │  ● = power points
├───────┼───────┼───────┤
│   ●   │       │   ●   │
├───────┼───────┼───────┤
│       │   ●   │       │
└───────┴───────┴───────┘
```

### Golden Ratio (1:1.618)

```
┌──────────────────────┬─────────┐
│                      │         │
│         A            │    B    │
│      (larger)        │(smaller)│
│                      │         │
└──────────────────────┴─────────┘
        A/B = (A+B)/A = 1.618
```

### Modular Grid

Create consistent spacing with a base unit:

```
Base unit: 8px
Margins: 3 units (24px)
Gutters: 2 units (16px)
Columns: Flexible
```

## Visual Weight

### Weight Factors

| Factor | Heavy | Light |
|--------|-------|-------|
| Size | Large | Small |
| Color | Dark/saturated | Light/desaturated |
| Position | Lower, right | Upper, left |
| Complexity | Detailed | Simple |
| Isolation | Surrounded by space | Crowded |

### Balancing Weight

**Symmetrical balance**: Mirror left/right or top/bottom
- Formal, stable, traditional

**Asymmetrical balance**: Different elements, equal visual weight
- Dynamic, interesting, modern

**Radial balance**: Elements arranged around center point
- Focused, ceremonial, mandala-like

## Hierarchy

### Creating Focus

1. **Size**: Largest element draws eye first
2. **Contrast**: High contrast areas dominate
3. **Color**: Saturated/warm colors advance
4. **Position**: Top-left to bottom-right reading pattern
5. **Isolation**: Space around element emphasizes it

### Visual Path

Guide the eye through the composition:

```
Entry point → Secondary element → Tertiary → Exit

[HERO IMAGE]
       ↓
   [headline]
       ↓
  [body text]
       ↓
   [call to action]
```

## Negative Space

### Active vs Passive

**Passive negative space**: Default background, unconsidered
**Active negative space**: Deliberately shaped, part of design

### Using Space

- Luxury = more space
- Urgency = less space
- Clarity = consistent space
- Tension = uneven space

### Minimum Margins

| Content Type | Minimum Margin |
|--------------|----------------|
| Text blocks | 1em |
| Images | 8px |
| Headlines | 1.5× line height |
| Edge of canvas | 5% of dimension |

## Color Composition

### Color Weight

- Dark colors appear heavier
- Warm colors appear closer
- Saturated colors appear larger
- Cool colors recede

### Color Balance

```
60% - Dominant color (background, large areas)
30% - Secondary color (supporting elements)
10% - Accent color (highlights, calls to action)
```

### Contrast for Readability

| Background | Text | Contrast Ratio |
|------------|------|----------------|
| White | Black | 21:1 (maximum) |
| White | Dark gray | 12:1 (excellent) |
| Light gray | Dark gray | 7:1 (good) |
| Off-white | Medium gray | 4.5:1 (minimum) |

## Common Mistakes

### Avoid

- Centering everything (lacks hierarchy)
- Equal spacing everywhere (monotonous)
- Too many focal points (confusing)
- Elements touching edges (cramped)
- Unintentional tangents (elements barely touching)

### Fix

- Create clear hierarchy with varied spacing
- Use rhythmic variation (small, medium, large gaps)
- Establish one primary focal point
- Maintain breathing room at edges
- Either overlap deliberately or separate clearly

## Quick Checks

Before finalizing:

- [ ] Is there a clear entry point?
- [ ] Does the eye flow through the composition?
- [ ] Is visual weight balanced?
- [ ] Is there enough negative space?
- [ ] Are all elements within margins?
- [ ] Is hierarchy clear?
- [ ] Does nothing accidentally overlap?
