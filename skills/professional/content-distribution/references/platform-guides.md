# Platform-Specific Guides

Tactical advice for each major distribution platform.

## Twitter/X

### Optimal Posting

| Metric | Recommendation |
|--------|----------------|
| Best times | 8-10am, 12-1pm, 5-6pm (audience timezone) |
| Frequency | 3-5 tweets/day max |
| Thread length | 5-10 tweets ideal |
| Character usage | 240-260 chars performs best |

### Content Types

| Type | Engagement | Effort |
|------|------------|--------|
| Thread | High | Medium |
| Single tweet + image | High | Low |
| Quote tweet with insight | Medium | Low |
| Poll | Medium | Low |
| Link only | Low | Low |

### Thread Structure

```
Tweet 1: Hook (surprising claim or question)
Tweet 2-N: Build your points
Final tweet: Summary + CTA

Pro tips:
- Number your tweets (1/7, 2/7...)
- End with "Follow for more [topic]"
- Self-reply to continue engagement
```

### Hashtag Strategy

- 1-2 hashtags maximum
- Place at end, not inline
- Use niche hashtags over broad ones
- Track which drive discovery

## LinkedIn

### Optimal Posting

| Metric | Recommendation |
|--------|----------------|
| Best times | Tue-Thu, 7-8am, 12pm, 5-6pm |
| Frequency | 2-4 posts/week |
| Post length | 1,200-1,500 chars for engagement |
| First line | Crucial - hook them |

### Content Types

| Type | Engagement | Effort |
|------|------------|--------|
| Personal story + insight | Very High | Medium |
| Carousel (PDF) | High | High |
| Native video | High | High |
| Text-only with hooks | High | Low |
| Link posts | Low | Low |

### Post Structure


### Engagement Tips

- Reply to every comment (boosts algorithm)
- Post and engage for 30 min after
- Comment on others' posts before/after your post
- Avoid external links (algorithm penalty)

## Newsletter (Substack/Email)

### Optimal Sending

| Metric | Recommendation |
|--------|----------------|
| Best days | Tue, Wed, Thu |
| Best times | 10am local, or 6pm |
| Frequency | Weekly or bi-weekly |
| Length | 500-1500 words |

### Email Structure

```
Subject: [Compelling hook, 40-50 chars]
Preview text: [Expands on subject, 90 chars]

Hi [First Name],

[Opening hook - personal, relevant, short]

[Main content - one clear theme]

[Key takeaway or insight]

[CTA - one clear action]

[Sign-off]
[Name]

P.S. [Secondary CTA or teaser]
```

### Subject Line Formulas

| Formula | Example |
|---------|---------|
| Number + benefit | "5 ways to improve X" |
| Question | "Are you making this mistake?" |
| How-to | "How I [achieved result]" |
| Contrarian | "Why [common advice] is wrong" |
| Curiosity gap | "The one thing I changed..." |

### Growth Tactics

- Cross-promote on social
- Guest posts/interviews
- Lead magnets (free resources)
- Referral programs
- Landing page optimization

## YouTube

### Optimal Publishing

| Metric | Recommendation |
|--------|----------------|
| Best days | Fri, Sat, Sun |
| Best times | 12-4pm local |
| Video length | 8-15 min sweet spot |
| Frequency | Weekly minimum |

### Video Structure

```
0:00-0:30  Hook (what they'll learn/gain)
0:30-2:00  Context (why this matters)
2:00-end   Content (deliver the promise)
Final 30s  CTA (subscribe, next video)

Include:
- Pattern interrupts every 2-3 min
- B-roll or visual changes
- Timestamps in description
```

### Optimization

| Element | Best Practice |
|---------|---------------|
| Title | 40-60 chars, keyword at start |
| Thumbnail | Face + text + contrast |
| Description | Keyword-rich first 100 chars |
| Tags | 5-10 relevant tags |
| Cards | Link to related videos |
| End screen | Subscribe + playlist |

## Personal Blog/Website

### SEO Basics

| Element | Recommendation |
|---------|----------------|
| Title tag | Primary keyword + compelling |
| Meta description | 150-160 chars, include keyword |
| URL slug | Short, keyword-rich |
| Headers | H1 once, H2-H4 for structure |
| Images | Alt text with keywords |

### Content Structure

```markdown
# Title (H1 - one per page)

[Opening paragraph - hook + what they'll learn]

## Section 1 (H2)
[Content with subheadings as needed]

### Subsection (H3)
[Detailed content]

## Section 2 (H2)
[Continue pattern]

## Conclusion
[Summary + CTA]
```

### Internal Linking

- Link to related posts (3-5 per article)
- Create pillar content + clusters
- Update old posts with new links
- Use descriptive anchor text

## Cross-Platform Repurposing

### From Blog Post

| Destination | Transformation |
|-------------|----------------|
| Twitter thread | Key points as tweets |
| LinkedIn post | Personal angle + insight |
| Newsletter | Summary + personal context |
| YouTube | Video explainer |
| Instagram carousel | Visual summary |

### From Video

| Destination | Transformation |
|-------------|----------------|
| Blog post | Transcript + formatting |
| Twitter clips | 30-60 sec highlights |
| LinkedIn | Key moments + text |
| Instagram Reels | Vertical cuts |
| Newsletter | Summary + link |

### Timing

```
Day 1: Original content (blog/video)
Day 3: Twitter thread
Day 5: LinkedIn adaptation
Day 7: Newsletter inclusion
Week 2: Instagram carousel
Month 2: Updated repost
```
