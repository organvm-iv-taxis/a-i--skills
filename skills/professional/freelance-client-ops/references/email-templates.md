# Client Communication Templates

Email templates for common freelance client scenarios.

## Initial Inquiry Response

### Template

```
Subject: Re: [Project Name] Inquiry

Hi [Name],

Thanks for reaching out about [project/need]. [Brief acknowledgment
of their situation].

I'd love to learn more about what you're looking for. Could you share:
- What's the main problem you're trying to solve?
- What does success look like for this project?
- What's your ideal timeline?
- Do you have a budget range in mind?

I have availability starting [timeframe]. If that works for you,
I'd suggest a quick 20-minute call to discuss further.

[Calendar link or time suggestions]

Looking forward to learning more.

Best,
[Your Name]
```

## Discovery Call Confirmation

### Template

```
Subject: Confirmed: Discovery Call [Date/Time]

Hi [Name],

Looking forward to our call on [Day, Date] at [Time] [Timezone].

Here's the meeting link: [Link]

To make the most of our time, it would help if you could think about:
- Your main goals for this project
- Any examples of work you admire
- Key stakeholders involved in decisions
- Must-have vs. nice-to-have requirements

If anything comes up, feel free to reschedule using [calendar link].

Talk soon,
[Your Name]
```

## Post-Discovery Follow-Up

### Template

```

Hi [Name],

**Your Goals:**
- [Goal 1]
- [Goal 2]

**Key Requirements:**
- [Requirement 1]
- [Requirement 2]

**Timeline:** [Target dates]

**Next Steps:**
I'll put together a proposal by [date] with scope, timeline, and
investment details.

Please let me know if I missed anything or if priorities have shifted.

Best,
[Your Name]
```

## Proposal Submission

### Template

```
Subject: Proposal: [Project Name]

Hi [Name],

Attached is my proposal for [project description in brief].

**Quick Summary:**
- Scope: [1-2 sentence overview]
- Timeline: [Duration]
- Investment: $[Amount]

The proposal includes [key elements]. Happy to walk through it on a
call if you'd prefer.

This proposal is valid for [14/30] days. If you have questions or
want to discuss any aspects, I'm available [suggest times].

Looking forward to potentially working together.

Best,
[Your Name]
```

chance to review it?

```

```
Subject: Re: Proposal: [Project Name] - Checking In

I wanted to follow up one more time on the proposal for [project].

If timing isn't right or you've decided to go in a different
direction, no worries at all—just let me know so I can close
out my notes.

If you're still interested but have questions or concerns, I'm
happy to chat.

Best,
[Your Name]

## Handling Scope Creep

### Template

```
Subject: Re: [Project Name] - Additional Request

Hi [Name],

Thanks for the additional ideas for [feature/change].

This falls outside our current scope, but I'd be happy to add it.
Here's what that would look like:

**Additional Scope:**
- [Description]

**Impact:**
- Additional time: [X] hours
- Additional cost: $[Amount]
- Timeline impact: [Extends by X days / No change]

**Options:**
1. Add this at the cost above
2. Swap it for [something currently in scope]
3. Add it to a Phase 2

Let me know which works best for you, and I'll prepare a change
order if needed.

Best,
[Your Name]
```

## Late Payment Follow-Up

### Template (Day 1 Past Due)

```
Subject: Invoice #[Number] - Payment Reminder

Hi [Name],

Just a friendly reminder that Invoice #[Number] for $[Amount] was
due on [Date].

If you've already sent payment, please disregard this message.
Otherwise, please let me know if you need the invoice resent or
have any questions.

Thanks,
[Your Name]
```

### Template (Day 14 Past Due)

```
Subject: Invoice #[Number] - Past Due Notice

Hi [Name],

Invoice #[Number] for $[Amount] is now [14] days past due.

Per our agreement, I'll need to pause work on [current tasks] until
the outstanding balance is resolved. Late fees may also apply per
our contract terms.

Please let me know the status or if there are any issues I should
be aware of.

Best,
[Your Name]
```

## Testimonial Request

### Template

```
Subject: Quick Favor?

Hi [Name],

Hope [project/product] is going well!

I'm building out my website and would love to feature our work
together. Would you mind sharing a brief testimonial?

A few sentences about:
- What we worked on
- What the experience was like
- Any results or outcomes

No pressure at all—and I'm happy to draft something for your
approval if that's easier.

Thanks for considering!

Best,
[Your Name]
```
