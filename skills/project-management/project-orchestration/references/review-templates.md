## Emergency Reset Review

For when you've fallen off the wagon entirely.

```markdown
## Emergency Reset: [Date]

### Current State
**How long since last review?** _____ weeks

**Feelings right now:**
[ ] Overwhelmed / [ ] Lost / [ ] Guilty / [ ] Numb / [ ] Other: _____

### Minimum Viable Recovery

**Step 1: Brain dump (10 min)**
Everything on your mind, no organization:




**Step 2: Identify ONE project (5 min)**
Which ONE thing, if you made progress on it today, would feel like a win?


**Step 3: Find the next action (5 min)**
What is the smallest possible action? (2-minute version)


**Step 4: Do it now**
[ ] Done

**Step 5: Schedule proper review**
When will you do a full weekly review?
Date/time: _____

### Recovery complete. You're back.
```
